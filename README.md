# Atelier Vanta

Prototype frontend créé pour tester Luixas comme agent UX/UI capable de produire une landing page premium déployable.

## Stack

- Vite
- React
- CSS plain

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Notes

- Form submission is simulated locally.
- No external backend or CRM integration.
- Visual assets are CSS-driven to keep the prototype lightweight.
- The page is designed as a practical UX/UI direction test, not a final client website.
